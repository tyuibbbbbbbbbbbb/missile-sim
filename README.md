# 🚀 Missile Launch Simulator

הדמיית שיגור טילים אינטראקטיבית עם מפה חיה.

## תכונות
- 🗺️ מפה אינטראקטיבית (Leaflet/OpenStreetMap)
- 🎯 בחירת יעד: איראן, תימן, לבנון
- 💣 בחירת ראש נפץ: קונבנציונלי, חודר בונקרים, תרמוברי
- 🚀 אנימציית טיסת טיל בזמן אמת (5-7 דקות)
- 💥 אנימציית פיצוץ בהגעה
- ⏱️ טיימר ספירה לאחור

## הרצה
פשוט פתח את `index.html` בדפדפן.

## ⚠️ הבהרה
**זוהי הדמיה בלבד למטרות בידור. אין לזה שום קשר למערכות צבאיות אמיתיות.**
